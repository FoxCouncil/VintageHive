Internet Explorer 3.x / 4.x / 5.x / 6.x

Internet Explorer 3.x / 4.x:

  1. Open View > Options (or Tools > Internet Options in IE5)
  2. Click the Connection tab
  3. Click Proxy Settings or LAN Settings
  4. Check Use a proxy server
  5. Click Advanced
  6. Set the following:
    Type Address
    HTTP {{ serverip }}:{{ port_http }}
    Secure (HTTPS) {{ serverip }}:{{ port_https }}
    FTP {{ serverip }}:{{ port_ftp }}
  7. Click OK on all dialogs

Internet Explorer 5.x / 6.x:

  1. Open the Tools menu, then select Internet Options
  2. Click on the Connections tab
  3. Click LAN Settings
  4. Check Use a proxy server for your LAN
  5. Click Advanced
  6. Set the same addresses as the table above
  7. Click OK on all dialogs

Netscape Navigator 3.x / 4.x

Netscape Navigator 3.x:

  1. Open Options > Network Preferences
  2. Click the Proxies tab
  3. Select Manual Proxy Configuration and click View...
  4. Set the following:
    Type Address Port
    HTTP Proxy {{ serverip }} {{ port_http }}
    Security Proxy {{ serverip }} {{ port_https }}
    FTP Proxy {{ serverip }} {{ port_ftp }}
  5. Click OK


Netscape Communicator 4.x:

  1. Open Edit > Preferences
  2. Navigate to Advanced > Proxies
  3. Select Manual Proxy Configuration and click View...
  4. Set the same addresses as the table above
  5. Click OK

NCSA Mosaic
  1. Open Options > Network Preferences
  2. Enable Use Proxy Server
  3. Set the following:
    Type Address Port
    HTTP Proxy {{ serverip }} {{ port_http }}
    FTP Proxy {{ serverip }} {{ port_ftp }}
  4. Click OK

Note: Mosaic does not support SSL/HTTPS. Only HTTP and FTP proxying will work.

Opera 3.x / 4.x
  1. Open File > Preferences
  2. Click the Network tab
  3. Click Proxy Servers
  4. Set the following:
    Type Address
    HTTP {{ serverip }}:{{ port_http }}
    HTTPS {{ serverip }}:{{ port_https }}
    FTP {{ serverip }}:{{ port_ftp }}
  5. Click OK

AOL Browser
  1. Open My AOL > Preferences
  2. Click the WWW icon
  3. Select the Connection tab
  4. Click Setup
  5. Enable Connect through proxy server
  6. Set HTTP Proxy: {{ serverip }} Port: {{ port_http }}
  7. Click OK

Compatibility Tips

Windows 3.1 / 95 Users:

  • If you can't type the . in the IP address, use the numeric keypad
  • Some browsers may require localhost instead of 127.0.0.1

SSL / HTTPS Support:

Browser SSL Support
Internet Explorer 3.x Limited — may need additional patches
Netscape 3+ Full SSL with certificate import
NCSA Mosaic No SSL support (HTTP only)
Opera 3+ Varies by version

Certificate Installation:

  1. Visit http://admin.hive.com/ca.crt in your browser
  2. Netscape: Select "Accept this Certificate Authority for Certifying network sites"
  3. Internet Explorer: Click "Install Certificate" and follow the wizard
  4. Opera: Save the file and import via File > Preferences > Security > Certificates

Known Issues:

  • Some browsers may show certificate warnings for HTTPS sites
  • FTP passive mode may not work in older browsers
  • JavaScript support varies significantly between browsers
  • Some browsers may require system proxy settings instead of browser settings
