Creating an Account

VintageHive user accounts are created through the Admin Panel.

  1. Open http://admin.hive.com in your browser
  2. Log in to the Admin Panel
  3. Navigate to User Management
  4. Enter a username and password, then click Create

Account Rules:

  • Usernames must be 3 to 8 characters long
  • Usernames can only contain letters and numbers (A–Z, 0–9)
  • Usernames are case-insensitive (Fox and fox are the same account)

Your username is used across all VintageHive services that require authentication. Your e-mail address will be username@hive.com.

Per-Service Login Requirements
Service Login Required? Notes
Web Browsing (HTTP Proxy) No Browse freely once proxy is configured
HTTPS Proxy (DialNine) No Works automatically with proxy settings
E-Mail (SMTP) Yes Must authenticate before sending mail
E-Mail (POP3) Yes Username and password required to check mail
IRC Optional Required only if using a registered nickname; unregistered nicks can connect freely
AIM / ICQ (OSCAR) Yes Account must exist before logging in with AIM or ICQ client
Telnet BBS No Connect and use immediately
FTP No Anonymous access; connect as user "anonymous"
