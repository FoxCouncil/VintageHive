A VintageHive user account is required to send and receive e-mail. Accounts can be created through the Admin Panel at http://admin.hive.com. Your e-mail address is username@hive.com (where "username" is your VintageHive account name).

Thunderbird / Modern clients: VintageHive supports automatic account configuration. If your DNS resolves autoconfig.hive.com to the VintageHive server, Thunderbird will auto-detect the correct IMAP/POP3/SMTP settings when you add a new account.

Outlook Express

Outlook Express 4.x / 5.x / 6.x:

  1. Open Tools > Accounts
  2. Click Add > Mail
  3. Enter your display name and click Next
  4. Enter your e-mail address: username@hive.com
  5. Set the mail servers:
    Setting Value
    Incoming mail server type POP3 or IMAP
    Incoming mail (POP3) {{ serverip }}
    Outgoing mail (SMTP) {{ serverip }}
  6. Enter your VintageHive username and password
  7. Click Finish

Setting custom ports:

  1. Open Tools > Accounts, select your account, click Properties
  2. Click the Advanced tab
  3. Set Outgoing mail (SMTP) port to: {{ port_smtp }}
  4. Set Incoming mail (POP3) port to: {{ port_pop3 }} (or IMAP port: {{ port_imap }})
  5. Click OK

Enable SMTP authentication:

  1. In account Properties, click the Servers tab
  2. Check My server requires authentication under Outgoing Mail Server
  3. Click Settings and select Use same settings as my incoming mail server
  4. Click OK

Eudora

Eudora 3.x / 4.x / 5.x:

  1. Open Tools > Options
  2. Select Getting Started in the left panel
  3. Fill in:
    Field Value
    POP Account username@{{ serverip }}
    Real Name Your display name
    Return Address username@hive.com
  4. Select Hosts in the left panel
  5. Set SMTP server to: {{ serverip }}
  6. Select Ports in the left panel (if available)
  7. Set POP3 port to: {{ port_pop3 }}
  8. Set SMTP port to: {{ port_smtp }}
  9. Click OK

SMTP Authentication (Eudora 5.x):

  1. In Tools > Options > Sending Mail
  2. Check SMTP server requires authentication
  3. Select Use POP account login

Netscape Mail

Netscape Communicator 4.x:

  1. Open Edit > Preferences
  2. Navigate to Mail & Groups > Mail Server
  3. Set the following:
    Field Value
    Outgoing mail (SMTP) server {{ serverip }}
    Incoming mail server {{ serverip }}
    Mail server type POP3
    Mail server user name username
  4. Click OK

Netscape Messenger 4.5+:

  1. Open Edit > Preferences
  2. Navigate to Mail & Newsgroups > Mail Servers
  3. Under Incoming Mail Servers, click Add
  4. Set Server Name: {{ serverip }}, Server Type: POP3, User Name: username
  5. Set Outgoing Mail Server to: {{ serverip }}
  6. Click OK

Note: Netscape's default SMTP port is 25 and POP3 port is 110. If VintageHive is using non-standard ports, Netscape 4.x may not support changing them directly. In that case, ensure VintageHive is configured to listen on the standard ports, or use a different client.

Mac OS Mail

Claris Emailer (Mac OS Classic):

  1. Open Setup > Internet Account Setup
  2. Click New and select POP
  3. Set POP Server: {{ serverip }}
  4. Set SMTP Server: {{ serverip }}
  5. Enter your VintageHive username and password
  6. Set your e-mail address to: username@hive.com
  7. Click OK

Eudora for Mac:

Follow the same steps as the Eudora section above. The Mac version uses the same menu structure and settings.

Outlook Express for Mac:

  1. Open Tools > Accounts
  2. Click New > POP
  3. Set Incoming POP server: {{ serverip }}
  4. Set Outgoing SMTP server: {{ serverip }}
  5. Enter your VintageHive username and password
  6. Set e-mail address: username@hive.com
  7. Click OK
