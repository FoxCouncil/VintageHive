FTP Server {{ serverip }} port {{ port_ftp }}

VintageHive's FTP server does not require authentication. Connect using any FTP client with the settings above. Passive mode is supported on ports 1900–1910.

WS_FTP

WS_FTP LE / Pro:

  1. Open WS_FTP — the Session Properties dialog will appear
  2. Click New to create a new session
  3. Fill in:
    Field Value
    Profile Name VintageHive
    Host Name/Address {{ serverip }}
    Host Type Automatic detect
    User ID anonymous
  4. Click the Advanced tab
  5. If the port is not the default (21), set Remote Port to: {{ port_ftp }}
  6. Ensure Passive transfers is checked
  7. Click OK, then Connect

CuteFTP

CuteFTP 2.x / 3.x:

  1. Open FTP > Site Manager (or press F4)
  2. Click New
  3. Fill in:
    Field Value
    Label VintageHive
    FTP Host Address {{ serverip }}
    FTP Site User Name anonymous
    FTP Site Password (leave blank)
    Port {{ port_ftp }}
  4. Click Connect

Enabling Passive Mode:

  1. In Site Manager, select the VintageHive entry
  2. Click Edit
  3. Check Use PASV mode (or under the Advanced tab)
  4. Click OK

Command Line FTP

Windows / DOS:

C:\> ftp {{ serverip }} {{ port_ftp }}
User: anonymous
Password: (press Enter)
ftp> dir
ftp> get filename.txt

Unix / Linux / Mac:

$ ftp {{ serverip }} {{ port_ftp }}
Name: anonymous
Password: (press Enter)
ftp> passive
ftp> ls
ftp> get filename.txt

Common FTP Commands:

Command Description
dir / ls List files in the current directory
cd dirname Change directory
get filename Download a file
put filename Upload a file
binary Switch to binary transfer mode
passive Toggle passive mode
quit Disconnect

ProtoWeb FTP Sites

VintageHive can proxy archived FTP sites from the ProtoWeb project. When you connect to an FTP host through VintageHive's proxy, it checks if that site is available in the ProtoWeb FTP archive and serves the archived content.

To browse ProtoWeb FTP archives, configure your browser's FTP proxy to point at VintageHive (see Client Settings), then visit FTP URLs directly in your browser, e.g.:

ftp://ftp.cdrom.com/
ftp://ftp.microsoft.com/
ftp://ftp.netscape.com/

You can also browse the full list of available ProtoWeb sites from the VintageHive intranet portal under the ProtoWeb Browser section.

Note: The list of available FTP sites is fetched from ProtoWeb and cached for 6 hours. Not all historical FTP sites are archived.
