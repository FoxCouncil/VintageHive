Telnet Server {{ serverip }} port {{ port_telnet }}

VintageHive's Telnet server is a text-mode BBS. No login is required — just connect and start typing commands. Terminal size is 80 columns by 24 rows.

Windows Telnet

Windows 95 / 98 / ME:

  1. Open Start > Run
  2. Type: telnet {{ serverip }} {{ port_telnet }}
  3. Click OK

Windows 3.1x:

  1. Open File Manager or Program Manager
  2. Run telnet.exe from the Windows directory
  3. Open Connect > Remote System
  4. Set Host Name: {{ serverip }}
  5. Set Port: {{ port_telnet }}
  6. Set Terminal Type: VT100
  7. Click Connect

From the DOS prompt:

C:\> telnet {{ serverip }} {{ port_telnet }}

PuTTY
  1. Open PuTTY
  2. In the Session settings:
    Field Value
    Host Name {{ serverip }}
    Port {{ port_telnet }}
    Connection type Telnet
  3. Optionally, type a name in Saved Sessions and click Save
  4. Click Open

Recommended PuTTY settings:

  • Terminal > Keyboard: set Backspace key to Control-H
  • Window > Columns: 80, Rows: 24

Unix / Mac

Open a terminal and run:

$ telnet {{ serverip }} {{ port_telnet }}

On Mac OS X (if telnet is not installed):

$ nc {{ serverip }} {{ port_telnet }}

Available Commands

Once connected, type any of the following commands at the prompt:

Command Description
help List all available commands
news Browse news headlines by category (Local, US, World, Tech, Science, Business, Entertainment, Sports, Health)
weather Current weather conditions and forecast; change your location and temperature units
riddle Interactive riddle game — you get 3 guesses per riddle
exit / quit Disconnect from the BBS
