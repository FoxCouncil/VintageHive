mIRC
  1. Open File > Options (or press Alt+O)
  2. In the tree, select Connect > Servers
  3. Click Add and fill in:
    Field Value
    Description VintageHive IRC
    IRC Server {{ serverip }}
    Port {{ port_irc }}
  4. If you have a VintageHive account, enter your password in the Password field
  5. Click OK, then click Connect to Server
  6. Once connected, type: /join #hive

Auto-join channels on connect:

  1. Open File > Options > Connect > Options > Perform
  2. Check Enable perform on connect
  3. Add the line: /join #hive

Pirch
  1. Open Server > Setup
  2. Click New to create a new server entry
  3. Fill in:
    Field Value
    Description VintageHive IRC
    Server {{ serverip }}
    Port {{ port_irc }}
  4. If you have a VintageHive account, enter your password in the Server Password field
  5. Click OK and then Connect

ircII / BitchX

Connect directly from the command line:

$ irc yournick {{ serverip }}:{{ port_irc }}

Or with BitchX:

$ BitchX -c #hive yournick {{ serverip }}:{{ port_irc }}

Once connected, use these commands:

Command Description
/join #hive Join the main channel
/list List all channels
/msg nick message Send a private message
/whois nick Look up user info
/quit Disconnect

Available Channels
Channel Topic
#hive Welcome to VintageHive IRC! — the main hangout
#vintage All things retro computing

Users can also create their own channels by joining a channel name that doesn't exist yet (e.g. /join #mychannel). User-created channels persist as long as they have members.

Authentication: Logging in is optional. If your nickname matches a registered VintageHive account, you must provide your password (via the server password field or /PASS) to use that nick. Unregistered nicknames can connect freely.
