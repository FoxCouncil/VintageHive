VintageHive includes a built-in NNTP news server with a curated collection of real Usenet newsgroup posts from the 1980s and 1990s, sourced from the Internet Archive's public Usenet collections. Point your favourite vintage newsreader at the server address above to browse groups and read articles. No account or password is needed — if your newsreader asks for credentials, enter anything and it will be accepted.

Outlook Express

Outlook Express 4.x / 5.x / 6.x:

  1. Open Tools > Accounts
  2. Click Add > News
  3. Enter your display name and click Next
  4. Enter your e-mail address and click Next
  5. Set the news server:
    Field Value
    News (NNTP) server {{ serverip }}
    My news server requires me to log on Unchecked
  6. Click Next, then Finish
  7. When asked "Would you like to download newsgroups?", click Yes

Setting a custom port:

  1. Open Tools > Accounts, select the news account, click Properties
  2. Click the Advanced tab
  3. Set News (NNTP) port to: {{ port_usenet }}
  4. Click OK

Reading Newsgroups:

  1. Click on the news server in the folder list
  2. Click Newsgroups to browse available groups
  3. Type a topic in the search box (e.g. comp.sys) to filter
  4. Select a group and click Subscribe, then Go to

Forte Agent / Free Agent

Forte Free Agent 1.x / Agent 2.x:

  1. Open Options > General Preferences
  2. Click the System tab
  3. Set the following:
    Field Value
    News Server {{ serverip }}
    Email Address Your e-mail address
    Full Name Your display name
  4. Click OK
  5. Open Options > News Server Properties
  6. Set the port to: {{ port_usenet }}
  7. Leave News server requires login unchecked
  8. Click OK
  9. Press F5 or Online > Refresh Groups List to download newsgroups

Subscribing to groups:

  1. Open Group > Show All Groups
  2. Type a search term to filter (e.g. alt.folklore)
  3. Double-click a group to subscribe

Netscape News

Netscape Communicator 4.x:

  1. Open Edit > Preferences
  2. Navigate to Mail & Newsgroups > Newsgroup Servers
  3. Click Add
  4. Set Server: {{ serverip }}
  5. Set Port: {{ port_usenet }}
  6. Leave Always use name and password unchecked
  7. Click OK

Netscape Navigator 3.x:

  1. Open Options > Mail and News Preferences
  2. Click the Servers tab
  3. Set News (NNTP) Server to: {{ serverip }}
  4. Click OK
  5. Open Window > Netscape News to access newsgroups

Note: Netscape 3.x uses the default NNTP port 119. If VintageHive is configured to use a different port, use Netscape Communicator 4.x or a dedicated newsreader like Forte Agent that supports custom ports.

Fetching Usenet Archives

VintageHive ships with a companion tool called UsenetCurator.exe that downloads real historical Usenet posts from the Internet Archive and prepares them for browsing.

Quick Start:

  1. Find UsenetCurator.exe in the same folder as VintageHive.exe
  2. Run it from the command line:

    UsenetCurator.exe

  3. Wait for the downloads to complete (this may take a while)
  4. Restart VintageHive — the new groups will appear automatically

Options:

Flag Description
--output <path> Where to write output files (default: data\usenet\)
--groups <list> Comma-separated list of groups to download
--max-per-group <n> Maximum articles per group

Downloaded mbox archives are cached locally. Subsequent runs skip already-downloaded files, making reruns much faster.
