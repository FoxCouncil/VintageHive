VintageHive's built-in NNTP server is planned for a future release. In the meantime, you can access Usenet through Eternal September — a free public news server that is accessible through VintageHive's HTTP proxy passthrough. You will need a free account at http://eternal-september.org to connect.

Outlook Express

Outlook Express 4.x / 5.x / 6.x:

  1. Open Tools > Accounts
  2. Click Add > News
  3. Enter your display name and click Next
  4. Enter your e-mail address and click Next
  5. Set the news server:
    Field Value
    News (NNTP) server news.eternal-september.org
    My news server requires me to log on Checked
  6. Enter your Eternal September username and password
  7. Click Finish
  8. When asked "Would you like to download newsgroups?", click Yes

Reading Newsgroups:

  1. Click on the news server in the folder list
  2. Click Newsgroups to browse available groups
  3. Type a topic in the search box (e.g. comp.sys) to filter
  4. Select a group and click Subscribe, then Go to

Forte Agent / Free Agent

Forte Free Agent 1.x / Agent 2.x:

  1. Open Options > General Preferences
  2. Click the System tab
  3. Set the following:
    Field Value
    News Server news.eternal-september.org
    Email Address Your e-mail address
    Full Name Your display name
  4. Click OK
  5. Open Options > News Server Properties
  6. Check News server requires login
  7. Enter your Eternal September username and password
  8. Click OK
  9. Press F5 or Online > Refresh Groups List to download newsgroups

Subscribing to groups:

  1. Open Group > Show All Groups
  2. Type a search term to filter (e.g. alt.folklore)
  3. Double-click a group to subscribe

Netscape News

Netscape Communicator 4.x:

  1. Open Edit > Preferences
  2. Navigate to Mail & Newsgroups > Newsgroup Servers
  3. Click Add
  4. Set Server: news.eternal-september.org
  5. Set Port: 119
  6. Check Always use name and password
  7. Click OK

Netscape Navigator 3.x:

  1. Open Options > Mail and News Preferences
  2. Click the Servers tab
  3. Set News (NNTP) Server to: news.eternal-september.org
  4. Click OK
  5. Open Window > Netscape News to access newsgroups

Note: Netscape 3.x does not support NNTP authentication natively. If Eternal September requires a login, use Netscape Communicator 4.x or a dedicated newsreader like Forte Agent.
